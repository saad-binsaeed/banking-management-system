# Banking Management System (CLI)

A simple Command Line Banking Management System built with Python using Object-Oriented Programming (OOP), File Handling, Authentication, and Transaction Management.

## Features

- Create Account
- Login Authentication
- Deposit Money
- Withdraw Money
- Check Balance
- Transaction History
- Data Persistence using File Handling
- Error Handling

## Technologies Used

- Python
- OOP
- File Handling
- Dictionaries
- Exception Handling

## How to Run

Clone the repository:

```bash
git clone https://github.com/yourusername/banking-management-system.git
```

Move into project folder:

```bash
cd banking-management-system
```

Run:

```bash
python main.py
```
